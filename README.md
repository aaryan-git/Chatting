AI Chat Assistant
An intelligent AI-powered chat application built with Next.js, TypeScript, and Gemini/OpenAI API, designed to provide real-time conversational responses with a modern UI.

🚀 Features
Real-time AI chat interface
Clean and responsive UI
Streaming AI responses
Conversation memory support
Modular component-based architecture
Fast and optimized with Next.js App Router
API integration with Gemini/OpenAI
Easy to customize and extend

🛠 Tech Stack
Frontend
Next.js 14+
React
TypeScript
Tailwind CSS
AI Integration
Gemini API / OpenAI API
UI Library
assistant-ui

Package Manager
npm / pnpm / yarn / bun
